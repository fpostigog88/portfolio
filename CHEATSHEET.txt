HOW TO FIND AND REPLACE IN NOTEPAD
----------------------------------
If you don't want to use RocketCake:
1. Right-click the HTML file
2. Open with > Notepad
3. Press Ctrl+H (Find and Replace)
4. Paste the text you want to change in "Find what"
5. Type your new text in "Replace with"
6. Click "Replace All"
7. Save the file
8. Refresh the page in your browser

IMPORTANT: Only change the text BETWEEN the quotes. Don't delete any < > or other symbols.
