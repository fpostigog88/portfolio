# Backlink Strategy

1. LinkedIn — highest recruiter and branded-query impact. Add website in contact, Featured, articles, and project posts.
2. GitHub — highest technical authority impact. Add website field, README backlinks, pinned repos, project descriptions.
3. YouTube — add website to channel and demo descriptions.
4. Michigan Ross — add profile/alumni link if available. Strong education/entity trust.
5. Medium — republish selected article excerpts linking to canonical website articles.
6. Substack — use only if maintained; link back to site.
7. Professional directories — use consistent full name/headshot/website.
8. Industry communities — publish practical posts tied to forecasting, automation, and decision systems.
