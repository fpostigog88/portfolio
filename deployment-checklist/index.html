External Authority and Search Deployment Checklist

This checklist shifts post-deployment effort away from more website edits and toward authority signals, backlinks, profile consistency, and indexing.

Ranked Highest-ROI Actions

  1. LinkedIn — highest impact: add felipepostigo.com to profile contact info, add the site to Featured, add Dell / AI forecasting / workflow automation project references, and use the same full name and headshot.
  2. Google Search Console: verify felipepostigo.com, submit https://felipepostigo.com/sitemap.xml, inspect the homepage, press kit, resume, and who-is page, then request indexing.
  3. Bing Webmaster Tools: import from Google Search Console or verify manually, submit sitemap.xml, and request indexing for homepage, press kit, who-is, about, and projects.
  4. GitHub: add website URL to profile, create or update profile README, pin major projects, and link project READMEs back to felipepostigo.com.
  5. YouTube: add website URL to channel profile and video descriptions, especially the workflow automation demo.
  6. Michigan Ross: check alumni directory, Michigan Ross alumni profile options, club pages, event speaker profiles, and career community pages for a possible website link.
  7. Other public profiles: keep Crunchbase, Wellfound, Maven, Substack/Medium, Kaggle, Hugging Face, personal GitHub repos, conference bios, and podcast bios consistent if used.

Google Search Console Exact Steps

  • Open Google Search Console.
  • Add property: https://felipepostigo.com/.
  • Choose DNS verification or HTML file verification.
  • Submit sitemap: https://felipepostigo.com/sitemap.xml.
  • Use URL Inspection for /, /who-is-felipe-postigo/, /press-kit/, /resume/, /projects/, /llms.txt, and /llms-full.txt.
  • Request indexing after deployment.

Bing Webmaster Tools Exact Steps

  • Open Bing Webmaster Tools.
  • Import the verified Google Search Console property or add https://felipepostigo.com/ manually.
  • Submit https://felipepostigo.com/sitemap.xml.
  • Use URL Submission for the homepage, press kit, who-is, resume, and projects pages.
